# 香港麻雀規則說明 (Hong Kong Style Mahjong Rules)

## 基本規則 (Basic Rules)

### 牌具
- 使用標準144張麻將牌（萬子、筒子、索子、字牌、花牌）
- 每人13張手牌，庄家14張

### 胡牌條件
- 必須組成4組面子（順子/刻子/槓子）+ 1對眼
- 雞胡（0番）可以胡牌
- 搶槓、槓上開花等特殊方式也可胡牌

### 計分方式說明

**出銃 (Discard Win)**
- 出銃者支付全部番數分數
- 例如：3番出銃 = 出銃者付8分

**自摸 (Self-Draw Win)**
- 每位玩家支付基本分數 × 1.5
- 例如：3番自摸 = 每位付12分，赢家收36分
- 包自摸：包家支付全部（12分 × 3 = 36分）

## 牌型番數 (Hand Types & Fan Values)

### 基本牌型 (Basic Hands)

| 牌型 | 番數 | 說明 |
|------|------|------|
| 雞胡 | 0番 | 最基礎的胡牌，無任何特殊組合 |
| 無花 | 1番 | 沒有花牌 |
| 門前清 | 1番 | 沒有吃、碰、槓別人的牌 |
| 自摸 | 1番 | 自己摸到胡牌（另計） |
| 河底撈魚 | 1番 | 摸最後一張牌而胡 |

### 花牌 (Flowers)

| 牌型 | 番數 | 說明 |
|------|------|------|
| 正花 | 1番 | 花牌數字與自己位置相同（東家拿春/梅） |
| 一花 | 1番 | 拿到1張花牌 |
| 兩花 | 2番 | 拿到2張花牌 |
| 三花 | 3番 | 拿到3張花牌 |
| 四花 | 4番 | 拿到4張花牌（全花） |

### 組合牌型 (Combination Hands)

| 牌型 | 番數 | 說明 |
|------|------|------|
| 一般高 | 1番 | 兩組相同的順子（如2組345萬） |
| 老少配 | 1番 | 123 + 789 同花色 |
| 喜相逢 | 1番 | 兩色相同的順子（如萬子+筒子都有345） |
| 連六 | 1番 | 123456 或 456789 同花色 |
| 碰碰胡 | 3番 | 4組全是刻子/槓子 |
| 對對糊 | 3番 | 同碰碰胡 |
| 七對 | 4番 | 7組不同的對子 |
| 豪華七對 | 7番 | 七對 + 有四張相同的牌（暗槓當兩對） |

### 花色牌型 (Suit Hands)

| 牌型 | 番數 | 說明 |
|------|------|------|
| 混一色 | 3番 | 只有一種花色 + 字牌 |
| 清一色 | 7番 | 只有一種花色，無字牌 |

### 么九、番子牌型 (Terminals & Honors)

| 牌型 | 番數 | 說明 |
|------|------|------|
| 混么九 | 4番 | 只有么九牌（1、9）+ 字牌 |
| 清么九 | 7番 | 只有么九牌，無字牌 |
| 小三元 | 5番 | 中發白有兩組刻子 + 一對眼 |
| 大三元 | 8番 | 中發白都是刻子 |
| 小四喜 | 10番 | 東南西北有兩組刻子 + 一對眼 |
| 大四喜 | 13番 | 東南西北都是刻子 |
| 字一色 | 13番 | 只有字牌（東南西北中發白） |

### 槓牌相關 (Kongs)

| 牌型 | 番數 | 說明 |
|------|------|------|
| 暗槓 | 1番 | 自己摸到四張相同的牌 |
| 明槓 | 0番 | 碰後再摸到第四張（部分規則計1番） |
| 槓上開花 | 3番 | 槓牌後補花牌而胡牌 |
| 搶槓 | 3番 | 別人加槓時胡那張牌 |

### 特殊牌型 (Special Hands)

| 牌型 | 番數 | 說明 |
|------|------|------|
| 十三么 | 13番 | 東南西北中發白 + 么九牌各一張 + 其中一張做眼 |
| 十八羅漢 | 13番 | 四組槓（明暗皆可）+ 胡牌 |
| 天胡 | 13番 | 莊家開牌即胡（部分規則不計） |
| 地胡 | 13番 | 閒家第一輪摸牌即胡 |

## 特殊規則 (Special Rules)

### 槓牌計番
- **暗槓**: 通常計1番
- **明槓**: 通常不計番或計1番
- **加槓**: 碰後再摸到第四張，可被搶槓

### 花牌處理
- 拿到花牌要立即補花（再摸一張）
- 正花（與自己位置對應的花牌）計1番
- 有玩家通常上限2番

### 包牌規則
- **包出銃**: 若玩家讓莊家/大牌家吃碰槓多次，導致其胡大牌，可能需要包賠
- **包自摸**: 同上，但包自摸者支付全部自摸分數

### 截胡
- 一炮多響時，通常按座位順序（逆時針）最靠近出銃者的人優先胡

## 計分示例 (Scoring Examples)

### 示例 1: 3番自摸
- 牌型: 碰碰胡 (3番)
- 基本分數: 8分
- 自摸倍數: ×1.5 = 12分/人
- **赢家收入**: 12 × 3 = 36分
- **每位輸家**: -12分

### 示例 2: 3番出銃
- 牌型: 碰碰胡 (3番)
- 基本分數: 8分
- **赢家收入**: +8分
- **出銃者**: -8分
- **其他兩家**: 0分

### 示例 3: 5番包自摸
- 牌型: 混一色 + 碰碰胡 (3+3=6番，部分規則只計5番)
- 基本分數: 24分
- 自摸倍數: ×1.5 = 36分/人
- 包自摸: 包家支付全部
- **赢家收入**: +36分
- **包家**: -36分
- **其他兩家**: 0分

### 示例 4: 高番數計算
- 牌型: 清一色 (7番) + 碰碰胡 (3番) = 10番
- 基本分數: 128分
- 自摸: 128 × 1.5 = 192分/人
- **赢家收入**: 192 × 3 = 576分
- **每位輸家**: -192分

## 參考資料

香港麻雀是香港最常見的麻雀玩法，特點是：
1. **番數制**: 簡單易計
2. **雞胡可胡**: 降低胡牌門檻
3. **自摸獎勵**: ×1.5倍鼓勵自摸
4. **花牌加分**: 增加遊戲變化
5. **大牌限制少**: 可以疊加番數

不同地區/家庭可能有細微差異，建議開局前確認規則。
