# Mahjong Ledger - Complete Feature Set

Based on the 麻將計數機 app, building all features:

## Supported Variants
1. 台灣麻將 (Taiwan Mahjong)
2. 日本麻雀 (Japanese/Riichi Mahjong) 
3. 香港麻雀 (Hong Kong Mahjong)
4. 港式台灣麻雀 (HK-Style Taiwan)
5. 碰槓牌/跑馬仔 (Pao Ma Zai)

## Core Features
- Multi-variant support with different rules
- One-click scoring (no manual number input)
- Undo/redo functionality
- Special cases: 包自摸, 一炮多響, 連莊拉莊
- Support 3-8 players with position swapping
- Settlement calculator (找數)
- Game statistics & player titles
- Results sharing (image export)
- Auto-scoring options

## Variant-Specific Rules
- Taiwan: 底/台, 連莊拉莊, 罰台, 詐胡, 門骰加倍
- Japanese: 原點/返點/馬點/不聽罰符, 立直, 延長賽, 荒牌流局
- HK-Style: 拉踢半, 投降, 擲圍骰, 槓賞罰
- HK: 全銃/半銃, Jackpot, 自摸倍數
- Pao Ma Zai: 獎馬, 包自摸, 富貴莊
